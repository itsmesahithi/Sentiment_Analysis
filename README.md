# Sentiment Analysis Project

## Overview
This project performs sentiment analysis on IMDB movie reviews using Natural Language Processing techniques.

## Dataset
IMDB Movie Reviews Dataset (Keras)

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- NLTK

## Model
- TF-IDF Vectorizer
- Logistic Regression

## Evaluation
- Accuracy Score
- Classification Report
- Confusion Matrix

## Key Insights
- TF-IDF with Logistic Regression gives strong baseline performance
- Text preprocessing improves model accuracy
- Negative and positive sentiments are well distinguished
